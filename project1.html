PROJECT OVERVIEW

The ECM (eCoachManager) Mobile System is an internship project focused on UX/UI design for mobile and iPad platforms. The goal was to improve user convenience and system accessibility. My responsibilities included wireframing, creating user journeys, prototyping, and designing user interfaces that are visually consistent and user-friendly. I conducted usability testing with real users and refined the design based on feedback. I also worked closely with the development team to ensure design feasibility and implemented responsive layouts to ensure smooth usage across both mobile phones and tablets.

PROBLEM STATEMENT

The original ECM (eCoachManager) system was designed primarily for desktop use, making it inconvenient for users to access while away from the office. This led to delays in retrieving important data and reduced overall workflow efficiency. Therefore, a mobile solution was needed to improve accessibility and support on-the-go usage.
