Project 01

ECM (eCoachManager) System for Mobile Xd

View Mobile Prototype View iPad Prototype

Project Overview

ECM (eCoachManager) is a transportation management system adapted for mobile use. This project involved creating a user-friendly interface that enables staff to manage bookings, schedules, and customer information on the go.

Problem Statement

The original desktop system was not optimized for mobile, leading to inefficiencies in on-site operations. Staff needed a responsive, portable solution to access data quickly. This redesign focuses on clarity, speed, and accessibility.
